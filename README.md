### executando o React
```sh
yarn start 
```

### executando o json-server

Json-serve é um servidor de texte para testar as rotas.

```sh
json-server server.json -p 3333 -w 
```

Servidor demorando 2 segundo na resposta
```sh
json-server server.json -p 3333 -w -d 2000
```
